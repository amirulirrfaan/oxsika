# Oxsika Alumni Website

## Description

The Oxsika Alumni Website is a platform designed to connect alumni, share updates, and engage with the Oxsika Ikups community. The website features sections for alumni news, events, fundraising, and more. It’s built using Next.js and Tailwind CSS for a modern and responsive design.

## Features

- **Hero Section**: Engaging hero section with a background image and call-to-action buttons.
- **About Us**: Information about Oxsika Ikups' history and mission.
- **News and Updates**: Latest news and updates for the alumni community.
- **Fundraising**: Details on current fundraising efforts and how to contribute.
- **Footer**: Includes links to various pages and social media.

## Technologies Used

- **Next.js**: React framework for server-side rendering and static site generation.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React**: Library for building user interfaces.
- **TypeScript**: Superset of JavaScript with static types for better development experience.
